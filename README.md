# GériaTimer

**Minuteur visuel d'exercices de maintien postural pour la rééducation gériatrique.**

Développé par **Sébastien THORENT**, masseur-kinésithérapeute D.E.  
Centre Hospitalier de Perpignan — SMR Polyvalent à orientation gériatrique.

---

## Présentation

GériaTimer est un outil gratuit conçu pour guider les patients âgés lors d'exercices de rééducation en salle. Il affiche en grand format sur un écran (TV, moniteur, PC) les consignes, un décompte et un personnage animé montrant le mouvement à réaliser.

L'interface est pensée pour des patients présentant des troubles cognitifs : code couleur clair entre exercice et repos, animations explicites, séquences entièrement programmables par le thérapeute.

## Fonctionnalités

- **Exercices animés** : debout, pointes de pieds, demi-squats, piétinement, montée de genoux, appui unipodal, monter/descendre une marche
- **Repos assis intégré** : le transfert assis-debout est animé automatiquement (le bonhomme s'assoit puis se relève)
- **Repos debout** : pour enchaîner des exercices debout sans s'asseoir
- **Programmes préchargés** : 12 séquences prêtes à l'emploi (assis-debout, équilibre, mixte...)
- **Personnalisation** : nom, durée, couleur et animation de chaque phase modifiables
- **Progression visuelle** : barre de pastilles montrant l'avancement dans la séance, numéro de série
- **Effet effort** : en fin d'exercice, le chiffre tremble et le personnage transpire
- **Plein écran** : mode immersif pour affichage sur TV de salle

## Utilisation

C'est un fichier HTML unique, autonome. Aucune installation requise.

1. Téléchargez `index.html` ou ouvrez directmeent ce lien https://seviiche.github.io/GeriaTimer/
3. Ouvrez-le dans n'importe quel navigateur (Chrome, Firefox, Edge, Safari)
4. Programmez votre séance ou choisissez un programme rapide
5. Lancez et affichez sur l'écran de la salle de rééducation

Fonctionne hors-ligne. Aucune donnée n'est envoyée nulle part.
